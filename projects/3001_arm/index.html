Unified Robotics III | Final Project

Vision-Based Pick & Place

Core Competencies

  • Kinematics: Implemented Forward (DH Convention) and Inverse Position Kinematics.
  • Trajectory: Developed Cubic and Quintic polynomial path generation for smooth motion.
  • Computer Vision: Created a HSV-based masking pipeline with distortion correction.
  • Velocity Control: Used Jacobian-based velocity kinematics for task-space movement.
  • System Logic: Designed a robust state machine to handle sorting and error recovery.

Overview

As part of Unified Robotics III, my team and I developed an autonomous targeting system for a 4-DOF robotic arm. The goal was to detect, identify, and sort colored spheres within a 3D workspace using an integrated overhead camera and mathematical modeling.

Figure 1: Diagram of the axes of motion and the DH Table of the OpenManipulator-X arm.

Kinematics & Motion Planning

The arm's navigation relies on a combination of geometric IK methods and kinematic decoupling. By separating position and orientation calculations, we reduced computational load and increased end-effector accuracy. To avoid the jerky motion common in basic robotic systems, we implemented Cubic Trajectory Generation, allowing for fluid transitions between task-space waypoints.

Computer Vision Pipeline

The vision system uses an overhead camera calibrated via MATLAB's Camera Calibration tool. Because the fisheye lens introduces significant distortion, we implemented a distortion correction algorithm that uses trigonometry and known constants (camera height and sphere size) to map pixels to real-world coordinates.

HSV Masking

Thresholding using HSV values to isolate sphere colors regardless of lighting shadows.

Centroid Logic

Pixel-to-coordinate mapping with size filtering to exclude noise from the task space.

System Architecture

The system operates on a State Machine architecture. The code separates vision (blocking) from motion (continuous updates). This ensures that once a target is identified and its position stored in the matrix, the arm executes its trajectory without being interrupted by transient visual noise or lighting changes.

Results & Conclusion

The final implementation successfully demonstrated high-fidelity sorting. The arm consistently identified targets, calculated correct inverse kinematics, and navigated to the appropriate bins. While tested with uniform spheres, the mathematical foundation of this project—specifically the Jacobian-based velocity control—is directly applicable to more advanced industrial robotics.
