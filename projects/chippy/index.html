3lb Combat Robot | April 2024 - Current

Chippy

Highlights

  • Autonomous ball collection and color-matching system
  • Vision and sensor-driven navigation
  • Ramp climbing with geared drivetrain
  • Modular Python codebase using VEX libraries
  • 1st place in final class competition

As part of WPI’s RBE 1001 course, my team and I designed and built an autonomous robot capable of collecting colored balls, identifying matching targets, and launching the balls through color-coded holes. The robot also successfully navigated a ramp as part of the challenge. Our project placed 1st in a class-wide competition.

Design

We designed a dual-conveyor system with custom 3D-printed flaps to collect and elevate balls. The upper conveyor acted as the launch mechanism, precisely timed for accuracy. The drivetrain featured a 4:1 gear ratio for torque, enabling our robot to reliably climb the ramp obstacle under load.

Sensors

Our robot utilized line sensors to know where to move, a vision sensor to align targets, an ultrasonic sensor to measure distances, and an inertial sensor to ensure accurate turns.

Programming

We programmed the robot using Python and the VEX Robotics API within Visual Studio Code. The system operated on a custom state machine architecture that transitioned between search, collection, aiming, and launch modes. Sensor feedback was modularized into reusable classes for clean navigation logic and target detection.

Reflection

This project taught me the importance of integrating mechanical design, sensor feedback, and software logic early in the development process. I also gained hands-on experience with autonomous behavior sequencing and collaboration under tight deadlines. Our robot’s performance was a result of extensive iteration and system-level thinking.
