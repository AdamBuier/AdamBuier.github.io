Ball Launcher Robot

RBE 1001 Project | Oct 2023 - Dec 2023

For my RBE 1001 class, a team and I were tasked with building an autonomous robot. This robot had to pick up colored balls, launch them through colored holes in a wall, and climb up a ramp. We competed against other classmates and placed 1st.

Design

The design for our robot used conveyor belts with flaps to lift and launch the balls. Additionally, we had a 4:1 gear ratio drive train which allowed us to climb the ramp.

Sensors

Our robot utilized line sensors to know where to move, a vision sensor to align targets, an ultrasonic sensor to measure distances, and an inertial sensor to ensure accurate turns.

Programming

Our code utilized Python and VEX libraries in Visual Studio Code. We used state machines to manage autonomous and manual modes. We also used custom classes to use sensor data for navigation and object detection.
